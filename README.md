# Снайперская подготовка — PWA-тесты

Веб-приложение (PWA) для повторения теории по снайперской подготовке.
Работает офлайн, ставится на домашний экран iPhone/Android, без App Store.

## Режимы
- **Экзамен** — 30 случайных вопросов, разбор ошибок в конце (без проходного порога).
- **Повторение — все темы** — бесконечный поток случайных вопросов с мгновенным фидбэком.
- **Темы** — повторение по конкретной теме.

Варианты ответов перемешиваются при каждом показе. Прогресс хранится локально (localStorage), без сервера.

## Структура
```
index.html          # разметка
css/style.css        # тёмная тактическая тема
js/app.js            # вся логика (3 режима, перемешивание, разбор)
data/questions.json  # база вопросов  ← редактируй здесь
manifest.json        # PWA-манифест
sw.js                # service worker (офлайн-кэш)
icons/               # иконки приложения
tmp/                 # вспомогательное (парсинг, генерация, вычитка) — не для деплоя
```

## Как редактировать вопросы
Правь `data/questions.json`. Формат:
```json
{ "q": "Текст вопроса?", "options": ["A", "Б", "В", "Г"], "correct": 2 }
```
`correct` — индекс правильного варианта (0 = первый). 3–4 варианта на вопрос.

После правок:
1. Проверь валидность: `python3 -c "import json; json.load(open('data/questions.json'))"`
2. Подними версию кэша в `sw.js` (`sniper-v1` → `sniper-v2`), иначе у пользователей останется старая версия.
3. Запушь — GitHub Pages обновится сам.

Удобный просмотр базы с пометкой ответов: `python3 tmp/dump_review.py` → `tmp/review.txt`.

## Локальный запуск
```
cd ~/sniping-quiz
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Деплой (GitHub Pages)
Репозиторий → Settings → Pages → Branch: `main`, папка `/ (root)`.
Ссылку вида `https://<логин>.github.io/sniping-quiz/` раздать пользователям.
На iPhone: открыть в Safari → «Поделиться» → «На экран Домой».
